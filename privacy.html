🔒 プライバシーポリシー

最終更新日: 2026年1月28日

1. はじめに

本プライバシーポリシーは、「カツミ・ヒロシ動画自動投稿システム」（以下「本アプリ」）が Google APIを通じて取得するユーザーデータの取り扱いについて説明します。

2. 収集する情報

本アプリは以下の情報にアクセスします：

  • YouTubeチャンネルへの動画アップロード権限
  • YouTubeチャンネルの基本情報（チャンネル名、ID等）

3. 情報の使用目的

取得した情報は以下の目的のみに使用されます：

  • ユーザーのYouTubeチャンネルへの動画アップロード
  • アップロードした動画のメタデータ設定（タイトル、説明文等）

4. 情報の共有

本アプリは、取得したユーザー情報を第三者と共有、販売、または貸与することは 一切ありません。

5. データの保存

認証情報（アクセストークン、リフレッシュトークン）は、ユーザーの ローカル環境またはセキュアなサーバー環境にのみ保存され、 暗号化された状態で管理されます。

6. Google API サービスの利用規約への準拠

本アプリのGoogle APIの使用は、 Google API サービス利用規約 に準拠しています。

7. ユーザーの権利

ユーザーは以下の権利を有します：

  • いつでもアプリへのアクセス権限を取り消すことができます
  • Google アカウント設定から本アプリの接続を解除できます
